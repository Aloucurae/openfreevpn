# OpenVPN Access Server With Unlimited Licence
##### Server Requirement: **Centos 7 x86_64**


### Installation

```sh
$ yum install wget -y
$ wget https://git.io/JtuWL && sed -i -e 's/\r$//' centos7.sh && chmod 755 centos7.sh && ./centos7.sh
$ node app
```
